# iTerm

## Keys

- ⌥← Send Escape Sequence Esc+ b

- ⌥→ Send Escape Sequence Esc+ f

- ⌘← Send Hex Code 0x01

- ⌘→ Send Hex Code 0x05

- ⌘←Delete Send Hex Code 0x15

- ⌥←Delete Send Hex Code 0x17

- ⌘Del→ Send Hex Code 0x0b

- ⌥Del→ Send Escape Sequence d
